# English Pronunciation Practice

Local React + Vite app that uses the browser SpeechSynthesis and SpeechRecognition APIs to practice speaking.

Run locally:

```bash
npm install
npm run dev
```

Notes:
- Works best in Chrome-derived browsers that support `SpeechRecognition`.
- For more accurate phoneme-level scoring consider integrating a pronunciation-scoring API (Google Cloud, Speechace, or other paid services).
